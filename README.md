## Keybindings

| Shortcuts                                   | Functionality        |
| ------------------------------------------- | -------------------- |
| `ESC`                     			      | Hide Panel           |
| `Tab`                     			      | Focus Last App       |
| `CMD + \`                     			  | Toggle Panel - Focus |
| `CMD + L`                     			  | Listen Source        |
| `CMD + S`                     			  | Swap Languages       |
| `CMD + Enter`                     		  | Fix Typo             |
